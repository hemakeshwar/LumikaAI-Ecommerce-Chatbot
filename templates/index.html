What is Lumika AI?

A custom-built intelligent chatbot for E-Commerce customer support using NLP techniques — TF-IDF vectorization and Cosine Similarity to match user queries to the best FAQ answers.

🔧

How It Works

User input is preprocessed — stopwords removed, typos corrected, synonyms expanded — then vectorized. Cosine similarity finds the closest FAQ match with confidence scoring.

♻️

Fully Customizable

Lumika AI adapts to any domain by swapping the dataset CSV. Healthcare, education, banking — just change the data and it works out of the box.

📚

NLP Concepts Used

Text preprocessing · Stopword removal · Tokenization · Typo correction · Synonym expansion · TF-IDF · Cosine Similarity · Intent detection · Regex pattern matching
